Keyboard Shortcuts

*ATENSION* Keyboard Shortcuts option is now available only Google Chrome beta.

If you use Google Chrome beta, you can configure Keyboard Shortcuts on the following chrome scheme URI.
Please open the URI in new tab/window via right click context menu, since chrome scheme URI is not clickable and not linkable.
chrome://extensions/configureCommands

Latest source code on github: https://github.com/yasuyk/Pinboard-Chrome-Extension.
